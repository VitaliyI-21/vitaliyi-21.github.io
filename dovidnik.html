Довідник з HTML та CSS

HTML — Основи

  • <!DOCTYPE html> — визначає тип документа як HTML5, обов'язковий для кожної HTML-сторінки.
  • <html> — кореневий елемент, що містить увесь HTML-код сторінки.
  • <head> — частина документа, що містить мета-інформацію, стилі, підключення шрифтів і скриптів.
  • <body> — основна частина сторінки, видима користувачу.
  • <h1>–<h6> — заголовки різних рівнів, де <h1> — найголовніший.
  • <p> — абзац тексту, зазвичай для довших пояснень чи описів.
  • <a href="..."> — гіперпосилання. Атрибут href задає адресу призначення.
  • <img src="..." alt="..."> — вставляє зображення, де alt — опис для читачів і SEO.
  • <br> — перенос рядка, не потребує закриваючого тегу.
  • <hr> — горизонтальна лінія, часто використовується для візуального поділу контенту.
  • <ul>, <ol>, <li> — створюють списки: маркований, нумерований, та елементи списку відповідно.
  • <div> — блочний контейнер для групування елементів або розділів сторінки.
  • <span> — рядковий контейнер, переважно для локального стилювання частин тексту.
  • <input> — інтерактивне поле для введення даних (текст, число, чекбокс тощо).
  • <button> — створює кнопку, яку можна натиснути для виконання дії.
  • <form> — контейнер для збору та відправлення даних на сервер або обробку скриптом.
  • HTML-коментар — <!-- коментар -->, не відображається в браузері, використовується для пояснень.

CSS — Основи

  • Селектор — визначає елемент, до якого застосовуються стилі (наприклад, div, .клас, #ідентифікатор).
  • color — встановлює колір тексту. Може бути заданий назвами, hex-кодом, rgb тощо.
  • background-color — колір фону елемента.
  • font-size — визначає розмір шрифту (в пікселях, em, rem тощо).
  • font-family — вказує набір шрифтів, наприклад Arial, 'Fira Code'.
  • margin — зовнішній відступ (зовні елемента).
  • padding — внутрішній відступ (всередині елемента).
  • border — рамка навколо елемента, може мати стиль (solid, dashed), товщину, колір.
  • width / height — задає ширину / висоту елемента (в px, %, vw, vh тощо).
  • text-align — вирівнює текст всередині елемента (left, center, right).
  • display — визначає тип відображення: block, inline, flex, grid тощо.
  • position — спосіб позиціонування: static (за замовчуванням), relative, absolute, fixed, sticky.
  • top, right, bottom, left — координати елемента при позиціонуванні.
  • overflow — регулює, що відбувається з вмістом, який виходить за межі блоку (visible, hidden, scroll, auto).
  • z-index — порядок накладання (чим більший — тим вище).
  • flexbox — модуль для побудови гнучких лінійних макетів: display: flex, justify-content, align-items.
  • grid — сучасна сіткова система компонування блоків: display: grid, grid-template-columns, grid-column тощо.
  • border-radius — закруглення кутів елементів.
  • :hover, :active, :focus — псевдокласи для взаємодії користувача.
  • @media — медіазапити для адаптації під розмір екрану.
  • transition — плавність зміни властивостей (час, ефект, затримка).
  • animation — анімації з ключовими кадрами (@keyframes).
  • visibility — визначає, чи елемент видимий: visible, hidden.
  • opacity — прозорість елемента (0 — повністю прозорий, 1 — повністю видимий).
  • box-shadow — тінь елемента.
  • cursor — вигляд курсора при наведенні (pointer, default, etc.).
  • content — використовують у ::before або ::after для вставки вмісту.
